Overview

Email aliasing is a default feature in Flectra which allows the user to fetch contents to ERP records directly from incoming emails. The feature is a type of automation. Since we can configure the E-mails to fetch contents directly. So here we are providing one of the new module to create leave request automatically from incoming email.

Here we can send leave request through our registered email id. Then it will create leave request in Flectra. We must ensure that our mail subject must start with 'LEAVE REQUEST ' and date format must be as '%d/%m/%Y'.
